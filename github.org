* CodeGen

1. Collect data from several sources:

| Format      | File format                    | Status  |
|-------------+--------------------------------+---------|
| source code | sources -> doxygen -> xml file | -       |
| uml model   | xmi file                       | -       |
| org-mode    | .org                           | initial |
| excel sheet | xml                            | -       |

2. Combine and process the data

3. Generate source code for
 - C#
 - C++
 - C 
 - Java
